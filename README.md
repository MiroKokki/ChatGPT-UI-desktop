# ChatGPT-UI-desktop
A simple python ChatGPT UI as a tkinter desktop app

## Usage
1. Downlaod all necessary files [zip file](https://github.com/MiroKokki/ChatGPT-UI-desktop/archive/refs/heads/main.zip) 
2. Install packages
    ```pip install openai```
    ```pip install tkinter```
3. Run main.py ```python main.py```
